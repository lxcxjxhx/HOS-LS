# Case Studies Index

## Overview

This directory contains real-world case studies demonstrating HOS-LS's effectiveness in identifying, analyzing, and remediating security vulnerabilities across different project types, sizes, and technology stacks.

## Data Classification

| Badge | Meaning |
|-------|---------|
| `ILLUSTRATIVE` | Simulated data based on realistic scenarios; not from actual customer engagement |
| `VERIFIED` | Data verified through real customer engagement or public vulnerability disclosure |
| `ANONYMIZED` | Real data with customer identity protected under NDA |

## How to Contribute

1. Copy [template.md](template.md) to create a new case study
2. Follow the naming convention: `NNN-short-title.md`
3. Submit via Pull Request to the `main` branch
4. Case studies will be reviewed by the security research team before merging

## Disclaimer

All illustrative case studies are clearly labeled and based on realistic scenarios constructed from publicly known vulnerability patterns and industry-standard benchmarks. They are intended for demonstration and educational purposes only.
